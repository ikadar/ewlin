## Layout Dimensions

| Element | Width | Tailwind Class |
|---------|-------|----------------|
| Sidebar | 56px | `w-14` |
| Jobs List | 288px | `w-72` |
| Job Details Panel | 288px | `w-72` |
| Date Strip | 48px | `w-12` |
| Timeline Column | 48px | `w-12` |
| Station Column (normal) | 240px | `w-60` |
| Station Column (collapsed) | 120px | `w-30` |

## Grid Constants

| Constant | Value | Description |
|----------|-------|-------------|
| Pixels per hour | 80px | Vertical scale |
| Start hour | 6:00 | Grid starts at 6 AM |
| Hours displayed | 24 | Full day visible |
| Snap interval | 30min | Drop positions snap to :00/:30 |

## Animation

| Property | Duration | Easing |
|----------|----------|--------|
| Tile transitions | 150ms | ease-out |
| Column collapse | 150ms | ease-out |
| Button hover | 100ms | default |

## Typography

- **Base font size:** 14px
- **Font weight:** 500 (medium)
- **Font smoothing:** antialiased
