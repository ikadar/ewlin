## Validation Feedback Colors

Visual feedback during drag operations:

| State | Color | Usage |
|-------|-------|-------|
| Valid | <span style={{background: '#22C55E', padding: '2px 12px', borderRadius: '4px'}}>&nbsp;</span> `#22C55E` | Drop is allowed |
| Invalid | <span style={{background: '#EF4444', padding: '2px 12px', borderRadius: '4px'}}>&nbsp;</span> `#EF4444` | Drop is blocked |
| Warning | <span style={{background: '#F97316', padding: '2px 12px', borderRadius: '4px'}}>&nbsp;</span> `#F97316` | Soft constraint (Plates pending) |
| Bypass | <span style={{background: '#F59E0B', padding: '2px 12px', borderRadius: '4px'}}>&nbsp;</span> `#F59E0B` | Alt+drop precedence bypass |

## Grid Constants

| Constant | Value | Description |
|----------|-------|-------------|
| Pixels per hour | 80px | Vertical scale |
| Start hour | 6:00 | Grid starts at 6 AM |
| Hours displayed | 24 | Full day visible |
| Snap interval | 30min | Drop positions snap to :00/:30 |
