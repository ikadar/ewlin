## Overview

Enhance the DateStrip with visual markers that provide at-a-glance understanding of task statuses, job timeline, and current viewport position. This transforms the DateStrip from a simple navigation tool into a comprehensive timeline overview.

---

## Prerequisites

- [x] `v0.3.46` released (Virtual Scrolling for Multi-Day Grid)

## Scope

### In Scope

**DateStrip Visual Markers:**

| Marker | Label | Description |
|--------|-------|-------------|
| **A** | Viewport Indicator | Light gray rectangle showing which portion of the day is visible in the grid |
| **B** | "Now" Line | Red horizontal line positioned within the viewport indicator based on current time |
| **C** | Exit Triangle | White triangle at workshop exit date (visible when job is selected) |
| **D** | Task Timeline | Dotted gray vertical line from earliest task to exit date |
| **E** | Task Markers | Colored markers for each scheduled task on the day |

**Task Marker Colors:**

| Status | Color | Description |
|--------|-------|-------------|
| Scheduled (future) | Gray (`zinc-500`) | Task scheduled but not yet started |
| Precedence conflict | Orange (`amber-500`) | Task has precedence violation |
| Late (past incomplete) | Red (`red-500`) | Past task not completed |
| Completed | Green (`emerald-500`) | Task completed |

### Out of Scope

- Interactive markers (clicking markers to navigate to task)
- Marker hover tooltips
- Multiple job comparison view

## Technical Notes

### Viewport Indicator (A)

The viewport indicator shows which vertical portion of the day is currently visible in the scheduling grid.

```
DateStrip Cell (h-10 = 40px)
┌─────────────────┐
│  Ma             │  ← Day abbrev
│  07             │  ← Day number
│ ┌─────────────┐ │
│ │░░░░░░░░░░░░░│ │  ← Viewport indicator (gray rectangle)
│ │░░──────────░│ │  ← "Now" line (red) if today
│ │░░░░░░░░░░░░░│ │
│ └─────────────┘ │
└─────────────────┘
```

**Calculation:**
```typescript
interface ViewportIndicatorProps {
  /** Total hours in the day (24) */
  hoursPerDay: number;
  /** Current grid scroll position (hour offset) */
  viewportStartHour: number;
  /** Viewport height in hours */
  viewportHours: number;
  /** Current time for "now" line (if today) */
  currentTime?: Date;
}

// Example: If viewport shows hours 8-16 of a 24-hour day
// indicator spans (8/24) to (16/24) = 33% to 67% of cell height
```

### Exit Triangle (C)

White triangle marker at the workshop exit date (departure date). Shows where the job must be completed by.

```
DateStrip Cell
┌─────────────────┐
│  Ve             │
│  10             │
│              ▽  │  ← Exit triangle (white)
│                 │
└─────────────────┘
```

### Task Timeline (D)

Dotted vertical line connecting the earliest scheduled task to the exit triangle.

```
DateStrip (multiple days)
┌─────────────────┐
│  Lu  05         │
│  ●              │  ← Task marker (first scheduled task)
│  ┆              │  ← Dotted line starts
├─────────────────┤
│  Ma  06         │
│  ●  ┆           │  ← Task + continuing line
├─────────────────┤
│  Me  07         │
│     ┆           │  ← No task, line continues
├─────────────────┤
│  Je  08  ▽      │  ← Exit triangle (line ends)
└─────────────────┘
```

### Task Markers (E)

Small horizontal colored lines indicating tasks scheduled on that day. Multiple tasks on the same day are stacked vertically.

```
DateStrip Cell with Task Markers
┌─────────────────┐
│  Ma             │
│  07             │
│ ────            │  ← Task marker (horizontal line)
│ ────            │  ← Another task marker (stacked)
│ ────            │  ← Third task marker
└─────────────────┘
```

```typescript
type TaskMarkerStatus = 'scheduled' | 'conflict' | 'late' | 'completed';

interface TaskMarkerProps {
  taskId: string;
  status: TaskMarkerStatus;
  verticalPosition: number; // Vertical stacking position for multiple tasks
}

// Status determination:
// - completed: assignment.isCompleted === true
// - late: !isCompleted && scheduledEnd < now
// - conflict: assignment has PrecedenceConflict
// - scheduled: default (future, no issues)
```

### Data Flow

```
App.tsx
  ├── gridScrollTop (pixels)
  ├── pixelsPerHour
  ├── viewportHeight
  ├── selectedJob (for exit date, tasks)
  └── assignments (for task status)
       │
       ▼
DateStrip
  ├── viewportStartHour = gridScrollTop / pixelsPerHour
  ├── viewportHours = viewportHeight / pixelsPerHour
  ├── departureDate (exit)
  └── taskMarkers: Array<{dayIndex, status, taskId}>
       │
       ▼
DateCell (per day)
  ├── ViewportIndicator (if this day is visible)
  ├── NowLine (if today, within viewport)
  ├── ExitTriangle (if departure date)
  └── TaskMarkers (if tasks scheduled)
```
