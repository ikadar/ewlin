## Overview

Replaces the plain text input for the Surfacage field in the Elements Table with a specialized `JcfSurfacageAutocomplete` component. Supports recto/verso DSL format (mat/mat, brillant/, UV/UV) with bidirectional DSL ↔ pretty display conversion, 10 preset suggestions with descriptions, session learning, and table navigation delegation.

This is the third field-specific autocomplete in Phase 4F, following the same wrapping pattern as v0.4.13 (Format) and v0.4.14 (Impression).

---

## Spec Source

- **§1.4 (Surfacage format)** from `reference/jcf/docs/implicit-logic-specification.md`
  - Format: `recto/verso` (e.g., `mat/mat`, `brillant/`, `UV/UV`)
  - Validation: must contain `/`
  - 10 presets: 5 R/V (mat/mat, satin/satin, brillant/brillant, UV/UV, dorure/dorure) + 5 recto-only (mat/, satin/, brillant/, UV/, dorure/)

## Technical Notes

### Surfacage DSL (§1.4)

| DSL | Pretty Display |
|-----|----------------|
| `mat/mat` | `pelli mat recto/verso` |
| `satin/satin` | `pelli satin recto/verso` |
| `brillant/brillant` | `pelli brillant recto/verso` |
| `UV/UV` | `vernis UV recto/verso` |
| `dorure/dorure` | `dorure recto/verso` |
| `mat/` | `pelli mat recto` |
| `satin/` | `pelli satin recto` |
| `brillant/` | `pelli brillant recto` |
| `UV/` | `vernis UV recto` |
| `dorure/` | `dorure recto` |
| Others | returned as-is |

### Validation

- Must contain `/` character (recto/verso separator)
- Free-text input allowed (custom surfacage values)

### Component Architecture

```
JcfElementsTable
  └── JcfSurfacageAutocomplete (wraps JcfAutocomplete)
        ├── editingValue state (internal, for editing)
        ├── isFocused state (toggle pretty/DSL display)
        ├── suggestions from SURFACAGE_PRESETS + session
        ├── onFocus: value → editingValue (DSL form)
        ├── onBlur: editingValue → store DSL + session learn
        └── session learning: learnSurfacage if new and valid (contains /)
```

### Suggestion Generation

SurfacagePreset[] mapped to Suggestion[]:
```typescript
{ label: "mat/mat", value: "mat/mat", category: "Pelli mat R/V" }
{ label: "UV/UV", value: "UV/UV", category: "Vernis UV R/V" }
```

Session-learned surfacages merged with mergeWithSession (session first).

### JcfElementsTable Integration

- Detect `row.key === 'surfacage'` in cell rendering
- Render JcfSurfacageAutocomplete with:
  - `value={element.surfacage}`
  - `onChange={(v) => handleCellChange(elementIndex, 'surfacage', v)}`
  - `presets={SURFACAGE_PRESETS}`
  - `sessionPresets={sessionLearning.surfacages}`
  - `onLearnPreset={sessionLearning.learnSurfacage}`
  - `onTabOut` / `onArrowNav` → delegate to cell navigation
  - `id={getCellId(elementIndex, rowIndex)}`
