## Overview

This release implements the Jobs List component (Left Panel), which displays all jobs organized by status. Problematic jobs (late, conflicts) appear at the top in a dedicated "Problèmes" section, followed by normal jobs in the "Travaux" section. Each job shows progress dots indicating task completion status.

## Scope

### In Scope

- **Jobs List container** - Fixed width panel with sections
- **Header** - Add Job button and search field
- **Problems section** - Late and conflict jobs with badges
- **Jobs section** - Normal job cards
- **Job cards** - Reference, client, description, progress dots
- **Selected state** - Highlighted job card
- **Search filtering** - Filter by reference, client, description

### Out of Scope

- Add Job modal functionality (button only, no modal)
- Keyboard navigation (deferred - shortcuts TBD)
- Full Redux state management (simple React state for selection)
