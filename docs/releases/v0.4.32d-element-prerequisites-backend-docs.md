## Overview

Backend API support and documentation for element-level prerequisites. This release completes the v0.4.32a work by adding PHP API changes and updating all documentation.

---

## Prerequisites

- [x] `v0.4.32a` released (Frontend: types, mock data, UI)

---

## Scope

### In Scope

**Backend (services/php-api):**
- Add `paperStatus`, `batStatus`, `plateStatus` fields to Element entity
- Remove prerequisite fields from Job entity
- Database migration for schema changes
- Update API endpoints (GET/POST/PUT for jobs and elements)
- Update OpenAPI documentation

**Documentation:**
- Update domain model (Job aggregate, Element entity)
- Update business rules (approval gates, paper procurement)
- Update initial data model (Job, Element, enums)
- Update architecture docs (service boundaries, interface contracts)

### Out of Scope

- Frontend changes (completed in v0.4.32a)
- Scheduler tile blocking visual (v0.4.32b)
- Forme status for die-cutting (v0.4.32c)
