> **Status:** ✅ Released (on ux-ui-development)

## Overview

This release implements the Sidebar component, the narrow navigation strip on the far left of the screen. It provides top-level navigation between main application views and is always visible.

---

## Prerequisites

- [x] v0.3.1 - Mock API released

---

## Scope

### In Scope

- **Sidebar container** - Fixed width strip with proper styling
- **Navigation icons** - Three Lucide icons for main views
- **Button states** - Active, inactive, and hover states
- **Accessibility** - ARIA labels and keyboard navigation

### Out of Scope

- Actual routing (will be added later)
- Settings functionality (disabled for MVP)
- Responsive/mobile layout

## Technical Notes

### Dimensions

| Property | Value |
|----------|-------|
| Width | 56px (w-14) |
| Height | Full viewport (h-screen) |
| Background | `bg-zinc-900/50` |
| Border | `border-r border-white/5` |

### Navigation Items

| Icon | Lucide Name | Purpose | MVP State |
|------|-------------|---------|-----------|
| Grid | `layout-grid` | Scheduling view | Active (default) |
| Calendar | `calendar` | Calendar overview | Inactive |
| Settings | `settings` | Application settings | Disabled |

### Button States

| State | Background | Text Color |
|-------|------------|------------|
| Inactive | transparent | `text-zinc-500` |
| Inactive + Hover | `bg-white/10` | `text-zinc-300` |
| Active | `bg-white/10` | `text-zinc-300` |
| Active + Hover | `bg-white/15` | `text-white` |
| Disabled | transparent | `text-zinc-700` |
