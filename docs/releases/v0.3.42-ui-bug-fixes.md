## Overview

Fix three UI bugs affecting user experience:
1. UnavailabilityOverlay not rendering on all days in multi-day grid
2. Job card content overflowing the sidebar container
3. Non-selected job tiles are not clickable when a job is selected

**Implements:** REQ-04, REQ-05, REQ-06 from [refactored-new-requirements-02-en.md](../ux-ui/tmp/refactored-new-requirements-02-en.md)

## Scope

### In Scope

**REQ-04: UnavailabilityOverlay Multi-Day Support**

The striped unavailability overlay (lunch breaks, non-operating hours) only appears on the first day in a multi-day grid. Other days show no indication of unavailable periods.

**Root cause:** `StationColumn` renders a single `UnavailabilityOverlay` using only the first day's schedule.

**Fix:** Render a separate overlay for each day in the multi-day grid, positioned at the correct Y offset.

**REQ-05: Job Card Overflow Fix**

Job cards in the sidebar can overflow their container when text content is too long.

**Root cause:** The header row in `JobCard` doesn't have proper overflow containment (`overflow-hidden`, `min-w-0`) to allow `truncate` to work on flex children.

**Fix:** Add `overflow-hidden min-w-0` to the header row and ensure all text elements use `truncate`.

**REQ-06: Non-Selected Tiles Should Be Clickable**

When a job is selected, tiles from other jobs cannot be clicked to switch selection. Expected: clicking any tile should select its job.

**Root cause:** `Tile.tsx` adds `pointer-events: none` to muted tiles when a job is selected (`isMutedBySelection`).

**Fix:** Remove `pointer-events: none` from the muted state. Keep visual muting (filter + opacity) but allow clicks.

### Out of Scope

- Multi-day grid infinite scrolling (REQ-09)
- DateStrip redesign (REQ-09)
- Layout reorganization (REQ-07)

## Technical Notes

### REQ-04: UnavailabilityOverlay Multi-Day Implementation

**Current state (buggy):**
```tsx
// StationColumn.tsx - renders ONE overlay for the entire column
const effectiveDayOfWeek = dayOfWeek ?? new Date().getDay();
const daySchedule = getDaySchedule(station, effectiveDayOfWeek);

<UnavailabilityOverlay
  daySchedule={daySchedule}
  startHour={startHour}
  hoursToDisplay={hoursToDisplay}
  pixelsPerHour={pixelsPerHour}
/>
```

**Fixed state:**
```tsx
// Calculate number of days in the grid
const numberOfDays = Math.ceil(hoursToDisplay / 24);
const gridStartDate = /* passed from parent */;

// Render overlay for each day
{Array.from({ length: numberOfDays }).map((_, dayIndex) => {
  const currentDate = addDays(gridStartDate, dayIndex);
  const dayOfWeekForDay = currentDate.getDay();
  const daySchedule = getDaySchedule(station, dayOfWeekForDay);
  const dayYOffset = dayIndex * 24 * pixelsPerHour;

  return (
    <UnavailabilityOverlay
      key={dayIndex}
      daySchedule={daySchedule}
      startHour={0}  // Each day starts at midnight
      hoursToDisplay={24}  // Each overlay covers one day
      pixelsPerHour={pixelsPerHour}
      yOffset={dayYOffset}  // Position at correct day
    />
  );
})}
```

**UnavailabilityOverlay changes:**
- Add optional `yOffset` prop to shift the overlay position
- Apply offset to all period positions

### REQ-05: Job Card Overflow Fix

**Issue 1: Text overflow**
```tsx
// Before: No overflow containment
<div className="flex items-center gap-2 mb-1">

// After: Proper flexbox truncation
<div className="flex items-center gap-2 mb-1 overflow-hidden min-w-0">
  <span className="font-mono text-[13px] shrink-0">{reference}</span>
  <span className="text-zinc-600 shrink-0">·</span>
  <span className="text-zinc-400 truncate min-w-0">{client}</span>
```

**Issue 2: Border clipping (w-full + margins overflow)**
```tsx
// Before: w-full (100%) + mx-2 (16px) = overflow!
className={`... text-left w-full`}

// After: calc accounts for margins
className={`... text-left w-[calc(100%-1rem)]`}
```

Key changes:
- Add `overflow-hidden min-w-0` to the flex container
- Add `shrink-0` to elements that shouldn't shrink (reference, dot)
- Add `min-w-0` to truncatable elements
- Replace `w-full` with `w-[calc(100%-1rem)]` to prevent margin overflow

### REQ-06: Non-Selected Tiles Clickable

**Current state (buggy):**
```tsx
// Tile.tsx line 216-220
const getMutingStyle = () => {
  if (isMuted) return { filter: 'saturate(0.2)', opacity: 0.6, pointerEvents: 'none' as const };
  if (isDragActive && !isDragging) return { pointerEvents: 'none' as const };
  return undefined;
};
```

**Fixed state:**
```tsx
const getMutingStyle = () => {
  // Keep visual muting but allow clicks
  if (isMuted) return { filter: 'saturate(0.2)', opacity: 0.6 };
  // Only disable pointer events during drag (to allow drop-through)
  if (isDragActive && !isDragging) return { pointerEvents: 'none' as const };
  return undefined;
};
```
