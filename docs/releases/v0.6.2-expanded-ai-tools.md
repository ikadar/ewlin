# v0.6.2 - Expanded AI Scheduler Tools

**Status:** In Progress
**Branch:** `feature/mcp-llm-scheduler`
**Date:** 2026-03-28

## Summary

Expands the AI Scheduler from 10 tools to 26 tools, enabling operators to perform virtually all scheduling operations via natural language. All new tools wrap existing PHP API endpoints — no backend changes required.

## New Tools (16 new)

### Read (2 new → 5 total)
| Tool | Description |
|------|-------------|
| `list_saved_schedules` | List available schedule snapshots |
| `get_job_dependencies` | Get dependency graph for a job |

### Write (14 new → 21 total)
| Tool | Description |
|------|-------------|
| `delete_job` | Permanently delete a job and all its tasks |
| `assign_task` | Assign a task to a station at a specific time |
| `reschedule_task` | Move a task to a different time/station |
| `split_task` | Split a task into two parts by ratio |
| `fuse_task` | Merge split task parts back together |
| `set_job_dependencies` | Set required jobs (replaces existing) |
| `remove_job_dependency` | Remove a single dependency link |
| `update_element_prerequisite` | Update BAT/paper/plates/formes status |
| `update_job` | Change deadline, shipper, or status |
| `update_outsourced_task_status` | Set outsourced task pending/progress/done |
| `save_schedule` | Save current schedule under a name |
| `load_schedule` | Load a saved schedule snapshot |
| `auto_place_job_asap` | ASAP placement for a single job |
| `auto_place_job_alap` | ALAP (deadline-aware) placement for a single job |
