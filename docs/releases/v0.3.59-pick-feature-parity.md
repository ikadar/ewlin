## Overview

Cette release restaure la parité de features entre le mode Drag & Drop classique et le nouveau mode Pick & Place (v0.3.54-v0.3.57). Deux features visuelles importantes n'avaient pas été portées vers Pick & Place:

1. **DryingTimeIndicator** (flèche jaune) - Montre la durée de séchage après les tâches d'impression
2. **Messages de validation** - Messages d'erreur en français lors d'un placement incompatible

## Technical Notes

### Issue 1: DryingTimeIndicator

Dans `StationColumn.tsx`, le `DryingTimeIndicator` a:
```tsx
isVisible={isDragging}  // ❌ Manque Pick mode
```

Doit devenir:
```tsx
isVisible={isDragging || isPickMode}
```

### Issue 2: Validation Messages

Dans `App.tsx`:
1. `pickValidation` state n'a pas de champ `message`
2. `PickPreview` ne reçoit pas le prop `validationMessage`

Solution:
1. Ajouter `message: string | null` à `pickValidation` state
2. Appeler `getPrimaryValidationMessage()` lors de la validation Pick
3. Passer le message à `PickPreview`
