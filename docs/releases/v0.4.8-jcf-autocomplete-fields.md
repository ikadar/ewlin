## Overview

Implements the generic JcfAutocomplete component and integrates it into the Job Header as Client and Template fields. This is the foundation autocomplete used by many later JCF releases (paper, format, impression, etc.). Includes keyboard navigation, text highlighting, lazy-loaded suggestions, category badges, and session learning for new clients.

## Scope

### In Scope

- Generic `JcfAutocomplete` component (reusable for all JCF fields)
- `highlightMatch` utility — bold+colored matching text in suggestions
- `useLazyLoadSuggestions` hook — initial 10 items, scroll to load more (max 25)
- Client field in JcfJobHeader with mock client data
- Template field in JcfJobHeader with client-filtered suggestions
- Session learning — new client names stored in session state on blur
- Focus chain: Client → Template → Intitulé
- Keyboard navigation: ↑↓ (navigate), Enter (select), Escape (close dropdown, not modal), Tab (close)
- Mock client and template data + API methods
- rem→px conversion for 13px base

### Out of Scope

- Auto-create client on save (v0.4.26 — save functionality)
- Template pre-fill of elements table (v0.4.27)
- Advanced autocomplete features: error tooltips, validation (later releases)

## Feature Checklist

### Component: JcfAutocomplete

- [x] **Suggestion interface** — `{ label, value, category? }`
- [x] **Dropdown open/close** — opens on focus and arrow keys, closes on click-outside / Escape / Tab
- [x] **Filtering** — case-insensitive substring match on label
- [x] **Highlighted item tracking** — keyboard nav and mouse hover
- [x] **Auto-scroll** — highlighted item scrolls into view on keyboard nav
- [x] **Selection** — Enter key or mouse click selects item, calls onChange + onSelect
- [x] **Escape handling** — stopPropagation to prevent modal close
- [x] **Text highlighting** — matching substring in bold/blue
- [x] **Category badges** — optional badge on right side of suggestion
- [x] **Lazy loading** — initial 10 items, scroll loads more, max 25, "scroll for more" indicator
- [x] **rem→px conversion** — all Tailwind classes at 13px base

### Component: highlightMatch

- [x] **Match highlighting** — bold+blue for matching text, bold+underline when highlighted
- [x] **No match** — returns plain text span
- [x] **Empty search** — returns plain text span

### Hook: useLazyLoadSuggestions

- [x] **Initial limit** — shows first 10 items
- [x] **Scroll to load** — loads more on scroll (threshold 50px from bottom)
- [x] **Max limit** — caps at 25 items
- [x] **Reset** — resets count when filter changes
- [x] **hasMore indicator** — boolean for "more items available"

### Integration: Client field in JcfJobHeader

- [x] **Client autocomplete** — w-[182px] (equivalent of w-72 at 13px base ≈ 0.8125×288)
- [x] **Mock client data** — 8 French client names
- [x] **Session learning** — on blur, new client values added to session suggestions with "nouveau" category
- [x] **Focus chain** — on client select → focus Template field

### Integration: Template field in JcfJobHeader

- [x] **Template autocomplete** — w-[182px], placeholder "Aucun"
- [x] **Client-filtered suggestions** — client-specific templates first, then universal
- [x] **Category badges** — client name or "universel"
- [x] **Mock template data** — 6 templates (some client-specific, some universal)
- [x] **Focus chain** — on template select → focus Intitulé field

### Mock data
