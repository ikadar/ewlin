## Overview

Complete the Pick & Place implementation by enabling picking from grid tiles (reschedule) and remove all drag & drop code. This release finalizes REQ-01 and eliminates the performance-heavy drag-and-drop interaction.

**Key changes:**
- Click on scheduled tiles to pick them (reschedule)
- Pulsating placeholder at original position
- Remove entire `dnd/` folder and `pragmatic-drag-and-drop` dependency
- Migrate E2E tests from drag to pick interactions

## Technical Notes

### Pick from Grid Behavior

```
Grid pick (reschedule existing tile):
┌─ Station A ─┐ ┌─ Station B ─┐ ┌─ Station C ─┐
│             │ │             │ │             │
│  ┌ ─ ─ ─ ┐  │ │             │ │             │
│  │PULSATE│  │ │    Ghost    │ │             │
│  └ ─ ─ ─ ┘  │ │   follows   │ │             │
│  (original) │ │   cursor    │ │             │
│             │ │             │ │             │
└─────────────┘ └─────────────┘ └─────────────┘
       ↑               ↑
  Placeholder     Can place anywhere
  at origin       (same or other station)

Key differences from sidebar pick:
- NO opacity change (all columns remain 100%)
- NO scroll (user is already at tile location)
- Placeholder shows at original position
```

### Pulsating Placeholder

```tsx
// In Tile.tsx - when isPicked is true
{isPicked && (
  <div
    className="absolute inset-0 border-2 border-dashed border-zinc-500
               rounded animate-pulse-opacity pointer-events-none"
    aria-hidden="true"
  />
)}
```

## Release Notes

## v0.3.57 – Pick & Place from Grid + Remove Drag & Drop

**Release Date:** 2026-01-22

### Summary

Complete Pick & Place implementation by enabling grid tile picking for rescheduling. Removes all drag & drop code for cleaner, more performant codebase.

### What's New

- **Pick from Grid:** Click on scheduled tiles to pick and reschedule them
- **Pulsating Placeholder:** Shows original position with animated dashed border
- **Grid Pick Behavior:** No column fade, no scroll (differs from sidebar pick)

### Removed

- **Drag & Drop:** Entire `dnd/` folder and `pragmatic-drag-and-drop` dependency removed
- **DragLayer:** No more drag preview overlay
- **Drop Targets:** Tiles no longer act as drop targets

### Technical Notes

- Pick & Place is now the only way to move tiles
- ~60 validations/second (drag) → 2 validations total (pick + place)
- Significant bundle size reduction from removing drag libraries
