## Overview

Complete the Pick & Place implementation by enabling picking from grid tiles (reschedule) and remove all drag & drop code. This release finalizes REQ-01 and eliminates the performance-heavy drag-and-drop interaction.

**Key changes:**
- Click on scheduled tiles to pick them (reschedule)
- Pulsating placeholder at original position
- Remove entire `dnd/` folder and `pragmatic-drag-and-drop` dependency
- Migrate E2E tests from drag to pick interactions

## Scope

### In Scope

**Pick from Grid:**
- Click handler on grid tiles (non-completed, non-outsourced)
- `isPicked` prop on Tile for placeholder rendering
- Pulsating placeholder at original position (uses CSS from v0.3.56)
- No opacity change for grid picks (all columns visible)
- No scroll for grid picks (user is already at tile location)
- Place on same or different station (within constraints)

**Remove Drag & Drop:**
- Delete `apps/web/src/dnd/` folder entirely
- Remove `pragmatic-drag-and-drop` from package.json
- Clean up App.tsx (remove DragStateProvider, drag handlers)
- Clean up Tile.tsx (remove draggable setup)
- Clean up StationColumn.tsx (remove drop target)
- Clean up TaskTile.tsx (remove draggable setup)
- Migrate E2E tests from drag to pick interactions

### Out of Scope

- Right-click context menu (v0.3.58)
- Job Details Panel fixed tile height (v0.3.59)
- Unavailability overlay SVG (v0.3.60)

## v0.3.57 вҖ“ Pick & Place from Grid + Remove Drag & Drop

**Release Date:** 2026-01-XX

### Summary

Complete Pick & Place implementation by enabling grid tile picking for rescheduling. Removes all drag & drop code for cleaner, more performant codebase.

### What's New

- **Pick from Grid:** Click on scheduled tiles to pick and reschedule them
- **Pulsating Placeholder:** Shows original position with animated dashed border

- E2E tests migrated from drag to pick interactions
