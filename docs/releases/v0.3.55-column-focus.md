## Overview

When picking an unscheduled task from the Job Details panel (sidebar), automatically scroll to and focus on the target station column. Non-target columns fade to 15% opacity to improve focus and prevent accidental interactions.

**Key behavior:**
- Sidebar picks: Scroll + fade non-target columns
- Grid picks (reschedule): No scroll, no fade (all columns remain visible)

### In Scope

- Save scroll position on sidebar pick (for cancel restoration)
- Scroll target station column to left edge (smooth, 300ms animation)

- Disable pointer events on non-target columns
- Restore scroll position on ESC cancel
- Restore opacity on place or cancel
- E2E tests for column focus behavior

### Out of Scope

- Column focus for grid picks (v0.3.57 - grid picks don't change column visibility)
- Pick from grid tiles (v0.3.57)
- Pulsating animation on original position (v0.3.56)
- Cursor state changes (v0.3.56)

## Technical Notes

### Scroll Behavior

```typescript
// On sidebar pick: Scroll target column to left edge
const stationIndex = snapshot.stations.findIndex(s => s.id === targetStationId);
const PADDING = 12;
const COLUMN_WIDTH = 240;
const GAP = 12;
const targetX = PADDING + stationIndex * (COLUMN_WIDTH + GAP);
gridRef.current.scrollToX(targetX, 'smooth');

// Save scroll position for cancel restoration
savedScrollRef.current = { x: currentScrollX, y: currentScrollY };
```

### Grid Spacer (Edge Case Fix)

To ensure the rightmost column can scroll to the left edge, a spacer div is added at the end of both the header and content areas:

```typescript
// In SchedulingGrid.tsx - after the last station/provider column
<div className="shrink-0" style={{ width: 'calc(100vw - 300px)' }} aria-hidden="true" />
```

This spacer ensures there's always enough content width for any column to scroll to the left edge of the viewport.

### Opacity Logic

```typescript
// In StationColumn.tsx
const getOpacityClass = () => {
  if (!isPicking) return '';
  if (pickSource !== 'sidebar') return ''; // Grid picks: no opacity change
  if (isPickTarget) return ''; // Target column: full opacity
  return 'opacity-15 pointer-events-none'; // Non-target: faded + disabled
};
```

### Cancel Restoration

```typescript
// On ESC key (when pickSource === 'sidebar')
if (savedScrollRef.current) {
  gridRef.current.scrollTo(savedScrollRef.current.x, savedScrollRef.current.y, 'smooth');
}
pickActions.cancelPick();
```
