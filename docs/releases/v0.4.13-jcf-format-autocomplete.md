## Overview

Replaces the plain text input for the Format field in the Elements Table with a specialized `JcfFormatAutocomplete` component. Supports ISO formats (A4, A3f, A5fi), custom dimensions (210x297), and composite formats (A3/A6) with bidirectional DSL ↔ pretty display conversion, dimension lookup, session learning, and table navigation delegation.

This is the first field-specific autocomplete in Phase 4F, consuming infrastructure from v0.4.11 (onTabOut/onArrowNav), v0.4.12 (session learning), and the base JcfAutocomplete component (v0.4.8).

## Technical Notes

### Format DSL (§1.5)

| Pattern | Example | Pretty Display |
|---------|---------|----------------|
| ISO | `A4` | `A4 — 210×297mm` |
| ISO fermé | `A4f` | `A4f — 420×297 ouvert / 210×297 fermé` |
| ISO fermé italienne | `A4fi` | `A4fi — 210×594 ouvert / 210×297 fermé` |
| Custom | `210x297` | `210×297mm` |
| Composite | `A3/A6` | `A3/A6 — 297×420 / 105×148` |

### Validation Regex

```typescript
const isoPattern = /^A([0-9]|10)(f|fi)?$/i;
const customPattern = /^\d+x\d+(f|fi)?$/i;
const compositePattern = /^(A([0-9]|10)|\d+x\d+)\/(A([0-9]|10)|\d+x\d+)$/i;
```

### Component Architecture

```
JcfElementsTable
  └── JcfFormatAutocomplete (wraps JcfAutocomplete)
        ├── editingValue state (internal, for editing)
        ├── isFocused state (toggle pretty/DSL display)
        ├── suggestions generated from ProductFormat[] + session
        ├── onFocus: value → editingValue (DSL form)
        ├── onBlur: editingValue → validate → normalize → store
        └── session learning: learnProductFormat if new and valid
```

### Suggestion Generation

ProductFormat[] mapped to Suggestion[]:
```typescript
{ label: "A4", value: "A4", category: "210×297mm" }
{ label: "A3", value: "A3", category: "297×420mm" }
```

Session-learned formats merged with mergeWithSession (session first).

### JcfAutocomplete `onFocus` Addition

```typescript
interface JcfAutocompleteProps {
  // ... existing props ...
  onFocus?: () => void;  // NEW: called when input receives focus
}
```

### JcfElementsTable Integration

- Import `useSessionLearning` hook (wire productFormats + learnProductFormat only)
- Import PRODUCT_FORMATS from mock data
- Detect `row.key === 'format'` in cell rendering
- Render JcfFormatAutocomplete with:
  - `value={element.format}`
  - `onChange={(v) => handleCellChange(elementIndex, 'format', v)}`
  - `formats={PRODUCT_FORMATS}`
  - `sessionPresets={sessionLearning.productFormats}`
  - `onLearnPreset={sessionLearning.learnProductFormat}`
  - `onTabOut` / `onArrowNav` → delegate to handleCellKeyDown
  - `id={getCellId(elementIndex, rowIndex)}`
