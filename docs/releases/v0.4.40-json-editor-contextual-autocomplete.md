## Overview

Add contextual autocomplete to the JSON editor in the Template Editor Modal. When editing template elements in JSON mode, the editor will detect which field the cursor is in and provide relevant suggestions based on reference data.

---

## Prerequisites

- [x] `v0.4.35` released (Dual-Mode Editor with CodeMirror)
- [x] `v0.4.39` released (SonarQube Integration)

---

## Scope

### In Scope

- Field detection based on cursor position in JSON
- Autocomplete suggestions for element fields:
  - `name` - Standard element names (COUV, INT, FEUILLET, etc.)
  - `format` - From reference data
  - `impression` - From reference data
  - `surfacage` - From reference data
  - `papier` - From reference data (paper types)
  - `sequence` - Workflow categories (Presse offset, Massicot, etc.)
- Trigger autocomplete when cursor enters autocompletable field
- Filter suggestions as user types

### Out of Scope

- Autocomplete for nested objects (e.g., `imposition.planche`)
- Real-time API suggestions (uses cached reference data)
- Inline documentation/tooltips for suggestions
