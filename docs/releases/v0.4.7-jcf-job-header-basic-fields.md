## Overview

Implements the basic input fields for the JCF Job Header — the horizontal form row at the top of the modal. This release covers the ID (readonly), Intitulé (text), Quantité (numeric), and Deadline (French date) fields with lifted state management. Client and Template autocomplete fields are deferred to v0.4.8.

## Feature Checklist

### Component: JcfJobHeader

- [ ] **Job ID field (readonly)**
  - Auto-generated in `J-YYYY-NNNN` format
  - Read-only input, monospace font, muted styling
  - tabIndex={-1} (not in tab order)
  - Width: fixed (w-28 equivalent)

- [ ] **Intitulé text field**
  - Free-text input for job description
  - Flex-grow to fill remaining space (min-width constraint)
  - Standard input styling with focus ring

- [ ] **Quantité numeric input**
  - Text input accepting numeric values
  - Right-aligned, monospace font
  - Fixed narrow width (w-24 equivalent)

- [ ] **Deadline date picker**
  - French format input: `jj/mm` (assumes current year) or `jj/mm/aaaa`
  - ISO 8601 internal storage (YYYY-MM-DD)
  - French display format on blur (jj/mm/aaaa)
  - Calendar icon (lucide-react `Calendar`)
  - Hidden native date picker overlay for calendar selection
  - Monospace font, fixed width (w-36 equivalent)

### Utilities

- [ ] **`parseFrenchDate(input: string): string`**
  - Parses `jj/mm` → ISO date (current year)
  - Parses `jj/mm/aaaa` → ISO date
  - Returns empty string for invalid input
  - Validates: month 1-12, day 1-31, actual date validity (no 31/02)

- [ ] **`formatToFrench(isoDate: string): string`**
  - Converts ISO date → `jj/mm/aaaa`
  - Returns empty string for empty input
  - Returns input unchanged for invalid ISO dates

- [ ] **`generateJobId(): string`**
  - Format: `J-{currentYear}-{random4digits}`
  - Zero-padded 4-digit random number

### Integration

- [ ] **State management in App.tsx**
  - `jcfJobId` state (generated once on modal open)
  - `jcfIntitule` state
  - `jcfQuantity` state
  - `jcfDeadline` state
  - Reset all on modal close
  - Pass through JcfModal to JcfJobHeader

- [ ] **JcfModal children**
  - JcfJobHeader rendered as first child in modal content area
  - Placeholder content below (existing placeholder from v0.4.6)

### Visual fidelity (rem→px at 13px base)

- [ ] Container: `p-[10px]`, `rounded-[3px]`, `gap-[10px]`
- [ ] Labels: `text-[10px]`, `mb-[3px]`
- [ ] Inputs: `rounded-[3px]`, `px-[7px]`, `py-[5px]`
- [ ] All sizes match reference/jcf with html { font-size: 13px }
