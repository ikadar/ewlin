## Overview

Move production prerequisite tracking from Job level to Element level. Each element tracks its own:
- **Paper** (Papier): Raw material availability
- **BAT** (Bon à Tirer): Client approval status
- **Plates** (Plaques): Offset printing plates preparation

This release adds the data model and basic UI in Job Details Panel. Blocking visual and tooltips are in v0.4.32b.

## Design

### New Status Types (packages/types/src/element.ts)

```typescript
/** Paper availability status */
export type PaperStatus =
  | 'none'        // No paper needed (assembly element or special case)
  | 'in_stock'    // Paper available in stock
  | 'to_order'    // Paper needs to be ordered
  | 'ordered'     // Paper has been ordered
  | 'delivered';  // Paper has been delivered

/** BAT (proof) approval status */
export type BatStatus =
  | 'none'            // No BAT required (assembly element or trusted client)
  | 'waiting_files'   // Waiting for client files
  | 'files_received'  // Files received from client
  | 'bat_sent'        // BAT sent to client for approval
  | 'bat_approved';   // BAT approved by client

/** Plates preparation status */
export type PlateStatus =
  | 'none'      // No plates needed (no offset action or assembly element)
  | 'to_make'   // Plates need to be made
  | 'ready';    // Plates are ready
```

```typescript
export interface Element {
```

### Offset Action Detection

An element "has offset action" if any of its tasks is assigned to a station with `categoryId === 'cat-offset'`.

```typescript
function hasOffsetAction(element: Element, tasks: Task[], stations: Station[]): boolean {
  return element.taskIds.some(taskId => {
    const task = tasks.find(t => t.id === taskId);
    if (!task || task.type !== 'Internal') return false;
    const station = stations.find(s => s.id === task.stationId);
    return station?.categoryId === 'cat-offset';
  });
}
```

### Assembly Element Detection

An element is an "assembly element" if it has no `spec` (no paper/format properties) OR if `spec.papier` is undefined.

```typescript
function isAssemblyElement(element: Element): boolean {
  return !element.spec || !element.spec.papier;
}
```
