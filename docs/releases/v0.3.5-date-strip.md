# Release v0.3.5 – Date Strip Component

> **Status:** ✅ Released
>

>

>

## Overview

This release implements the Date Strip component, a narrow column for day-based navigation. It displays a list of days with French abbreviations (Lu, Ma, Me, etc.) and highlights today with an amber background. Click to jump to a specific date in the grid.

---

## Prerequisites

- [x] v0.3.4 - Job Details Panel released

---

## Scope

### In Scope

- **DateStrip container** - Fixed width (w-12 / 48px), independent scroll
- **DateCell component** - Day abbreviation + day number per row
- **Today highlight** - Amber background for current day
- **Click to jump** - Navigate to specific date (callback only for MVP)
- **French day abbreviations** - Lu, Ma, Me, Je, Ve, Sa, Di

### Out of Scope

- Hover 2s → auto-jump (requires drag state awareness)
- Departure date highlight (requires job selection state)
- Grid scroll synchronization (requires grid component)
- Has-tiles indicators

## Technical Notes

### Dimensions

| Property | Value |
|----------|-------|
| Width | 48px (w-12) |
| Height | Full viewport height |
| Background | `bg-zinc-950` |
| Border | `border-r border-white/5` |
| Cell height | 40px (h-10) |

### French Day Abbreviations

| Day | Abbreviation |
|-----|--------------|
| Monday | Lu |
| Tuesday | Ma |
| Wednesday | Me |
| Thursday | Je |
| Friday | Ve |
| Saturday | Sa |
| Sunday | Di |

### Visual States

| State | Classes |
|-------|---------|
| Normal | `text-zinc-500`, `border-b border-white/5`, `hover:bg-white/5` |
| Today | `text-amber-200`, `bg-amber-500/15`, `border-b border-amber-500/30` |
