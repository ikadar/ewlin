## Overview

Introduce the Pick & Place interaction pattern for placing unscheduled tasks from the Job Details panel sidebar. This is the first step in replacing drag-and-drop with a more performant two-click interaction.

**Why Pick & Place?**
- Drag-and-drop: ~60 validations/second (mousemove events)
- Pick & Place: 2 validations total (pick + place)

This release focuses on **sidebar picking only** (unscheduled tasks). Grid tile picking will be implemented in v0.3.57.

## Technical Notes

### Architecture: Dual Rendering Streams

```
Stream 1: Position (60fps)     - RAF loop → direct DOM manipulation
Stream 2: Validation (100ms)   - Throttled, CSS transition masks latency
```

### Key Design Decisions

1. **Ghost position via useRef + RAF** - No React re-renders during mouse movement
2. **Portal-based ghost rendering** - Ghost tile rendered outside normal DOM hierarchy
3. **Validation throttled to 100ms** - Early-exit when same 15-min slot
4. **pickSource tracking** - Distinguish 'sidebar' vs 'grid' picks for later column focus logic
