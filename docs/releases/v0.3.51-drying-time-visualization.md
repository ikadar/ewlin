## Overview

During drag operations, visualize the drying time period with a yellow arrow and "End of drying" label. This helps users understand why the earliest start constraint (purple line) is positioned where it is - because printing tasks require 4 hours of drying time after completion.

**Implements:** Visual enhancement for precedence constraint understanding (related to REQ-04)

## Technical Notes

### Visual Design

Based on the provided mockup:

```
Predecessor task (printing station)
┌──────────────────┐
│   VS-001 · ...   │
│                  │
└──────────────────┘ ← Task end (e.g., 7:30h)
        │
        │  ← Yellow arrow (vertical line with arrowhead)
        │
        │    "End of
        │     drying"  ← Label
        ▼
- - - - - - - - - - - ← Dashed line (e.g., 11:30h = 7:30 + 4h)
```

### Components to Create

1. **DryingTimeIndicator component**
   - Yellow vertical line (arrow) from task end to drying end
   - Arrowhead at bottom
   - Dashed horizontal line at drying end
   - "End of drying" text label

### Positioning

```typescript
// Calculate drying indicator position
const predecessorEndY = timeToYPosition(predecessorEndTime, ...);
const dryingEndY = timeToYPosition(dryingEndTime, ...); // +4 hours

// Arrow spans from predecessorEndY to dryingEndY
const arrowHeight = dryingEndY - predecessorEndY;
```

### When to Show

```typescript
// Show drying indicator when:
// 1. Dragging a task
// 2. Task has a scheduled predecessor
// 3. There is drying time required (dry time > 0)
const showDryingIndicator =
  isDragging &&
  predecessorAssignment &&
  hasDryTime(snapshot, predecessorAssignment);
```
