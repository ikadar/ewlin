## Overview

Adds keyboard-driven cell navigation to the JCF Elements Table, enabling Tab/Shift+Tab vertical traversal, Alt+Arrow Excel-style movement with circular wrap, and Enter/Escape cell interaction. This transforms the Elements Table from mouse-only to keyboard-first editing, matching the reference JCF's navigation behavior.

## Scope

### In Scope

- **Tab / Shift+Tab navigation**: Move down/up within a column; overflow to next/previous column
- **Alt+Arrow navigation**: Alt+↓/↑/→/← with circular wrapping
- **Enter key**: Confirm current cell value and move to next cell (same as Tab)
- **Escape key**: Blur current cell (return focus to table container)
- **Cell ID system**: Unique IDs for programmatic focus (`cell-{elementIndex}-{rowIndex}`)
- **Focus visual indicator**: Cells already have `:focus` border style — ensure it works with programmatic focus

### Out of Scope

- Dropdown navigation (§8.2) — no autocomplete fields exist yet (v0.4.11+)
- Cmd+S / Ctrl+S save — no save functionality yet (v0.4.26)
- `onTabOut` / `onArrowNav` delegation from autocomplete fields (v0.4.11+)

## Technical Notes

### Navigation Model

The table has a 2D grid of cells addressed by `(elementIndex, rowIndex)`:

```
           ELT (0)    ELEM2 (1)   ELEM3 (2)
Row 0:     [cell]     [cell]      [cell]       ← precedences
Row 1:     [cell]     [cell]      [cell]       ← quantite
...
Row 11:    [cell]     [cell]      [cell]       ← sequence
```

### Tab Behavior (Vertical-First)

- **Tab**: Move to `(elementIndex, rowIndex + 1)`. At last row → `(elementIndex + 1, 0)`. At last cell → native Tab exits table.
- **Shift+Tab**: Move to `(elementIndex, rowIndex - 1)`. At first row → `(elementIndex - 1, lastRow)`. At first cell → native Shift+Tab exits table.

### Alt+Arrow Behavior (Circular Wrap)

- **Alt+↓**: `(elementIndex, (rowIndex + 1) % rowCount)` — wraps to top
- **Alt+↑**: `(elementIndex, (rowIndex - 1 + rowCount) % rowCount)` — wraps to bottom
- **Alt+→**: `((elementIndex + 1) % elementCount, rowIndex)` — wraps to first column
- **Alt+←**: `((elementIndex - 1 + elementCount) % elementCount, rowIndex)` — wraps to last column

### Cell ID Convention

```
id="cell-{elementIndex}-{rowIndex}"
```

- `elementIndex`: 0-based index within `elements` array
- `rowIndex`: 0-based index within the combined `rows` array (baseRows + sequenceRow = 12 rows)

### Focus Management

- Imperative DOM focus via `document.getElementById(id)?.focus()`
- No React ref map needed — IDs are deterministic from grid position
- `focusCell(elementIndex, rowIndex)` utility function

### Enter / Escape in Textareas

- **Enter in textarea fields** (commentaires, sequence): Allow newlines as usual (no navigation)
- **Enter in text inputs**: Move to next cell (same behavior as Tab)
- **Escape**: Blur current field
