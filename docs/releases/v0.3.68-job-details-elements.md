## Overview

Refactors the JobDetailsPanel to display tasks grouped by element. Each element section shows the suffix as header with prerequisite elements indicated by a Workflow icon.

---

## Prerequisites

- [ ] `v0.3.67` released (Element mock generators)

---

## Scope

### In Scope

- Group tasks by element in JobDetailsPanel
- Element section headers with suffix
- Workflow icon + prerequisite suffixes for dependent elements
- Maintain existing task interactions (Pick & Place, jump to grid)

### Out of Scope

- Tile badge (v0.3.69)
- Precedence logic changes (v0.3.70)

### Element Header Format

- Suffix in uppercase as section title
- For elements with prerequisites: Workflow icon + space-separated prerequisite suffixes (lowercase)
- Aligned to the right
