## Overview

Az első lépés a Production Flow Dashboard megvalósításában: teljes vizuális táblázat read-only adatmegjelenítéssel. Nincs interakció — sem szűrés, sem listbox, sem expand/collapse. A release célja, hogy a komplex táblázat vizuális struktúrája (frozen oszlopok, badge-ek, ring+dot SVG indikátorok, multi-element aggregált megjelenítés) helyesen jelenjen meg még az interaktív logika hozzáadása előtt.

## Technical Notes

- **Mock adatok:** v0.5.15 kizárólag statikus fixture adatokból dolgozik. Az API integráció v0.5.16+ scope.
- **Dark color tokens:** A spec rgb(15,15,15)–rgb(52,52,52) palettája nem pontosan egyezik a meglévő Tailwind zinc-* osztályokkal. A `dark.*` custom tokeneket hozzá kell adni a `tailwind.config.js`-hez (a mockup.html Tailwind config alapján).
- **Station progress % (deferred):** A ring+dot SVG arc-ok progress értékeit a fixture adatokban explicit adjuk meg (a spec 5.3-ban leírt állapotok alapján); az API mező kérdése (K3.1) halasztva.
- **"Late" állapot (deferred):** A fixture adatokban explicit `late` állapotot állítunk be (spec 5.3: job 00078 Ronde → Off. = late, job 00117 → Off. = late); a definíció API oldalon halasztva (K3.2).
- **Fonts:** Inter és JetBrains Mono már konfigurálva az `index.css`-ben.
- **Table min-width:** `min-w-[1440px]` a mockup alapján.
- **Sub-row interaktivitás:** v0.5.15-ben a sub-row-ok nem jelennek meg (minden multi-element sor összecsukva).
