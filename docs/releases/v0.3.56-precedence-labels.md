## Overview

Ajout de labels contextuels sur les lignes de précédence pendant le drag/pick. Ces labels indiquent la tâche contraignante et l'heure effective (incluant le drying time), donnant un contexte immédiat à l'utilisateur sans avoir à consulter le panneau détails.

**Origine:** Party Mode discussion (2026-01-15) - amélioration UX des contraintes de précédence.

- [x] `v0.3.55` released (Pick & Place Visual Feedback)

## Scope

### In Scope

- Label au-dessus de la ligne "earliest" (borne minimum)
- Label en-dessous de la ligne "latest" (borne maximum)
- Inclusion du drying time dans l'heure affichée
- Format court et iconique : `↑ {TaskName} · {HH:MM}`
- Affichage uniquement pendant drag/pick/quick placement
- Style translucide avec fond pour lisibilité

### Out of Scope

- Affichage de la chaîne complète de précédences (seulement direct)
- Labels permanents (seulement pendant interaction)
- Localisation avancée (format heure selon locale seulement)

## Technical Notes

### Format des labels

```
Borne minimum (earliest):
┌─────────────────────────────┐
│  ↑ Impression · 16:30       │  ← au-dessus de la ligne verte
├─────────────────────────────┤
│  ═══════════════════════════│  ← ligne verte (earliest)
└─────────────────────────────┘

Borne maximum (latest):
┌─────────────────────────────┐
│  ═══════════════════════════│  ← ligne rouge (latest)
├─────────────────────────────┤
│  ↓ Livraison · 18:00        │  ← en-dessous de la ligne rouge
└─────────────────────────────┘
```

### Calcul de l'heure effective

```typescript
// Pour earliest: heure de fin du prédécesseur + drying time
const effectiveTime = predecessorEndTime + dryingTime;

// Pour latest: heure de début du successeur - durée de la tâche courante
const effectiveTime = successorStartTime - currentTaskDuration;
```

### Props à ajouter à PrecedenceLines

```typescript
interface PrecedenceLinesProps {
  earliestY: number | null;
  latestY: number | null;
  isVisible: boolean;
  // New props for labels:
  earliestLabel?: {
    taskName: string;
    time: string; // Format HH:MM
  };
  latestLabel?: {
    taskName: string;
    time: string;
  };
}
```
