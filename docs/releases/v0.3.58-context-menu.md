## Overview

Add a right-click context menu on tiles that provides quick access to common actions. The menu provides an alternative to the visible swap buttons for a cleaner interface, especially useful for power users.

**Key changes:**
- Right-click on tile opens context menu at cursor position
- Menu options: View details, Toggle completion, Move up, Move down
- Auto-positioning to avoid viewport edges
- Close on click outside, ESC, or scroll

## Scope

### In Scope

**Context Menu Component:**
- `TileContextMenu.tsx` - portal-based component (z-index: 9999)
- Position at cursor (x, y) coordinates
- Auto-flip near right/bottom viewport edges
- French labels for all menu options

**Menu Options:**
| Option | Icon | Label (FR) | Action |
|--------|------|------------|--------|
| View details | Eye | Voir détails | Select job → show Job Details panel |
| Toggle completion | Square/CheckSquare | Marquer terminé / Marquer non terminé | Toggle `isCompleted` status |
| Move up | ChevronUp | Déplacer vers le haut | Swap with tile above |
| Move down | ChevronDown | Déplacer vers le bas | Swap with tile below |

**Close Triggers:**
- Click outside menu
- Press ESC key
- Scroll event (position would be stale)
- Click on menu item (after action)

**Constraints:**
- Only works on tiles (not on empty grid cells)
- Move up/down disabled if no adjacent tile exists
- No "Pick" option (pick is via single click on tile)

### Out of Scope

- Context menu on empty grid cells (not in this release)
- Custom keyboard shortcuts for menu items
- Job Details Panel fixed tile height (v0.3.59)
- Unavailability overlay SVG (v0.3.60)

## Technical Notes

### Menu Positioning Algorithm

```
┌───────────────────────────────────────────────────────┐
│                      Viewport                          │
│                                                        │
│      Normal position:          Near right edge:        │
│      ┌─ Menu ─┐                        ┌─ Menu ─┐      │
│      │ Item 1 │←cursor          cursor→│ Item 1 │      │
│      │ Item 2 │                        │ Item 2 │      │
│      │ Item 3 │                        │ Item 3 │      │
│      └────────┘                        └────────┘      │
│                                                        │
│      Near bottom edge:         Near corner:            │
│      ┌─ Menu ─┐                        ┌─ Menu ─┐      │
│      │ Item 3 │                        │ Item 3 │      │
│      │ Item 2 │                        │ Item 2 │      │
│      │ Item 1 │←cursor          cursor→│ Item 1 │      │
│      └────────┘                        └────────┘      │
└───────────────────────────────────────────────────────┘
```

### Component Structure

```tsx
// Portal-based rendering for z-index control
<Portal>
  <div
    className="fixed z-[9999] bg-zinc-800 rounded-lg shadow-xl border border-zinc-700"
    style={{ top: position.y, left: position.x }}
  >
    <MenuItem icon={Eye} label="Voir détails" onClick={onViewDetails} />
    <MenuItem icon={Square} label="Marquer terminé" onClick={onToggleComplete} />
    <Separator />
    <MenuItem icon={ChevronUp} label="Déplacer vers le haut" onClick={onMoveUp} disabled={!canMoveUp} />
    <MenuItem icon={ChevronDown} label="Déplacer vers le bas" onClick={onMoveDown} disabled={!canMoveDown} />
  </div>
</Portal>
```

### State Management

```tsx
// In App.tsx or SchedulingGrid
const [contextMenu, setContextMenu] = useState<{
  x: number;
  y: number;
  assignmentId: string;
  canMoveUp: boolean;
  canMoveDown: boolean;
} | null>(null);

// Right-click handler in Tile
const handleContextMenu = (e: React.MouseEvent) => {
  e.preventDefault();
  onContextMenu?.(e.clientX, e.clientY, assignment.id);
};
```
