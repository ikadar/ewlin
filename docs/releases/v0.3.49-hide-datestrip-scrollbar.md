## Technical Notes

### CSS Implementation

The scrollbar will be hidden using cross-browser CSS:

```css
/* Webkit (Chrome, Safari, Edge Chromium) */
.scrollbar-hide::-webkit-scrollbar {
  display: none;
}

/* Firefox */
.scrollbar-hide {
  scrollbar-width: none;
}

/* IE/Edge Legacy */
.scrollbar-hide {
  -ms-overflow-style: none;
}
```

### Tailwind CSS Approach

Since Tailwind CSS 3.x doesn't include a built-in `scrollbar-hide` utility, we'll add the CSS directly to the component using Tailwind's arbitrary value syntax or inline styles.

**Option A: Arbitrary CSS in className (Tailwind 3.x)**
```tsx
className="overflow-y-auto [&::-webkit-scrollbar]:hidden [-ms-overflow-style:none] [scrollbar-width:none]"
```

**Option B: CSS module or global styles**
Add to a CSS file with Tailwind's `@layer` directive.

We'll use Option A for simplicity since it's a single component change.
