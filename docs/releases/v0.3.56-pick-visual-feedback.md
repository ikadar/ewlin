## Overview

Enhance Pick & Place visual feedback with global grabbing cursor during pick mode, pulsating animation CSS (preparation for v0.3.57 grid picks), and validation throttle optimization.

**Key improvements:**
- Global `cursor: grabbing` during pick mode (like dragging)
- CSS pulsating animation for placeholder (used in v0.3.57)
- Validation early-exit when cursor stays in same 15-min slot

## Technical Notes

### Global Cursor During Pick Mode

```css
/* In index.css */
body.pick-mode-active {
  cursor: grabbing !important;
}

body.pick-mode-active * {
  cursor: grabbing !important;
}
```

```typescript
// In App.tsx - toggle body class when picking
useEffect(() => {
  if (isPicking) {
    document.body.classList.add('pick-mode-active');
  } else {
    document.body.classList.remove('pick-mode-active');
  }
  return () => document.body.classList.remove('pick-mode-active');
}, [isPicking]);
```

### Pulsating Animation (CSS)

```css
/* For v0.3.57 grid picks - placeholder at original position */
@keyframes pulse-opacity {
  0%, 100% {
    opacity: 1;
  }
  50% {
    opacity: 0.7;
  }
}

.animate-pulse-opacity {
  animation: pulse-opacity 1.5s ease-in-out infinite;
}
```

### Validation Throttle (Early-Exit)

```typescript
// Track last validated slot to skip redundant validations
const lastPickSlotRef = useRef<string | null>(null);

const handlePickMouseMove = useCallback((...) => {
  // Calculate slot from Y position
  const slotKey = `${stationId}-${Math.floor(snappedTileTop / (pixelsPerHour / 4))}`;

  // Early-exit: same slot, skip validation
  if (slotKey === lastPickSlotRef.current) {
    // Still update ghost position for smooth movement
    pickActions.updateGhostPosition(clientX, clientY);
    return;
  }

  lastPickSlotRef.current = slotKey;
  // ... run full validation
}, [...]);
```
