## Overview

Adds the ability to edit an existing job's metadata (client, description, deadline) through the JCF modal. A "Modifier" button on the Job Details Panel opens the JCF modal in edit mode, pre-filled with the selected job's data. Saving updates the existing job instead of creating a new one.

## Scope

### In Scope

- "Modifier" button on Job Details Panel header

- JCF modal edit mode (pre-fill, read-only job ID, different title/save label)

### Out of Scope

- Task-level editing (sequence, DSL)
