## Overview

Refonte majeure de l'UX des tuiles pour optimiser l'interface pour les power users (5h+/jour). Les actions secondaires sont déplacées vers un menu contextuel (clic droit), les tuiles affichent leur vraie taille proportionnelle, et un tooltip compense le manque d'espace sur les petites tuiles.

**Contexte utilisateur:** Power users passant 5h+ par jour sur l'outil. La discoverability est moins importante que l'efficacité et la propreté de l'interface.

## Scope

### In Scope

- **Context Menu:** Menu contextuel (clic droit) avec actions : View details, Mark as done, Move up, Move down
- **Checkbox Styling:** Checkbox carrée et plus grande pour la complétion (inline, cliquable)
- **True Tile Sizes:** Tuiles à taille proportionnelle réelle même à zoom faible (pas de hauteur minimale)
- **Smart Tooltip:** Tooltip au hover pour les tuiles trop petites (affiche job reference, client, station, durée)
- **Remove Visual Buttons:** Suppression des boutons œil et flèches (déplacés vers menu contextuel)

### Out of Scope

- Keyboard shortcuts pour actions du menu contextuel (future enhancement)
- Animations de menu (garder simple pour MVP)
- Touch/mobile support (desktop power users)

## Technical Notes

### Context Menu Implementation

Option A: **Radix UI ContextMenu** (recommended)
- Production-ready, accessible
- Handles positioning, keyboard nav, focus management
- Already used pattern in many React apps

Option B: **Custom implementation**
- Lighter weight
- Full control
- More work for accessibility

**Decision:** Start with custom implementation for simplicity, migrate to Radix if needed.

### Tile Height Calculation

Current behavior:
```tsx
// Minimum height to fit content
const totalHeight = Math.max(40, minutesToPixels(spanMinutes, pixelsPerHour));
```

New behavior:
```tsx
// True proportional height, no minimum
const totalHeight = minutesToPixels(spanMinutes, pixelsPerHour);
```

### Tooltip Trigger Logic

Show tooltip when:
1. Tile height < threshold (e.g., 32px)
2. OR content would overflow
3. AND mouse is hovering

Tooltip content:
```
┌────────────────────────┐
│ JOB-123 · Client ABC   │
│ Station: Offset        │
│ 1h30 (setup: 15min)    │
│ ☐ Non terminé          │
└────────────────────────┘
```
