## Overview

Redesign the DateStrip component with infinite scroll, focused day centering, and improved visual states. The current day indicator will change from amber background to a thin line (similar to the grid's "now" line), and a new "focused day" concept will be introduced to show which day is currently visible in the grid.

**Implements:** REQ-09 from [new-requirements-02.md](../ux-ui/tmp/new-requirements-02.md)

## Technical Notes

### REQ-09.1: Infinite Scroll Implementation

**Current implementation:**
- Fixed `dayCount` prop (default 21 days)
- Static array of dates generated on mount

**New implementation:**
- Remove `dayCount` dependency for rendering
- Use virtual scrolling or large buffer (e.g., 365 days)
- Scroll position initialized to show current day

**Alternative approach (simpler):**
- Keep large fixed buffer (e.g., 365 days: 180 before + 185 after today)
- No true infinite scroll, but sufficient for practical use
- Avoids complexity of dynamic date loading

### REQ-09.2: Focused Day Centering

**Implementation approach:**
1. Add `focusedDate` prop to DateStrip
2. Calculate focused date from grid scroll position
3. Scroll DateStrip to center the focused date
4. Highlight focused date cell

**Scroll sync logic:**
```typescript
// In App.tsx - calculate focused date from grid scroll
const handleGridScroll = (scrollTop: number) => {
  const hoursFromTop = scrollTop / pixelsPerHour;
  const focusedDate = addHours(gridStartDate, hoursFromTop + 12); // Center of viewport
  setFocusedDate(focusedDate);
};

// In DateStrip - scroll to center focused date
useEffect(() => {
  if (focusedDate && containerRef.current) {
    const dateIndex = differenceInDays(focusedDate, startDate);
    const cellHeight = 40; // h-10
    const containerHeight = containerRef.current.clientHeight;
    const targetScroll = (dateIndex * cellHeight) - (containerHeight / 2) + (cellHeight / 2);
    containerRef.current.scrollTop = targetScroll;
  }
}, [focusedDate, startDate]);
```

### REQ-09.3: Visual States

**Current day (today) - NEW styling:**
```tsx
// Replace amber background with thin line indicator
<button className="h-10 w-full relative ...">
  {/* Thin red line like grid's "now" line */}
  {isToday && (
    <div className="absolute left-1 right-1 top-1/2 h-0.5 bg-red-500 -translate-y-1/2" />
  )}
  <span>...</span>
</button>
```

**Focused day - NEW state:**
```tsx
// Highlighted background/border for focused day
{isFocused && (
  <div className="absolute inset-0 bg-white/10 border border-white/20 rounded" />
)}
```
