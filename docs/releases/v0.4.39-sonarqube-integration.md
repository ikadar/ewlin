## Release Notes

### v0.4.39 – SonarQube Integration

**Release Date:** 2026-02-02

### Summary

This release integrates SonarQube for code quality analysis and fixes the majority of CRITICAL and MAJOR issues. The codebase maintainability is significantly improved with reduced cognitive complexity and cleaner patterns.

### What's New

- SonarQube integration with `/sonar` command
- CRITICAL issues reduced from 17 to 0
- MAJOR issues reduced from 24 to 4
- Extracted helper functions for better readability
- Native buttons replacing role="button" divs where possible

### Known Issues

- 4 S6819 (accessibility) issues remain unfixable:
  - `Tile.tsx`: Contains nested interactive elements (SwapButtons)
  - `JcfModal.tsx`, `JcfTemplateEditorModal.tsx`, `StationColumn.tsx`: Modal backdrops with role="button"
  - These require `div` with `role="button"` due to HTML constraints (`<button>` cannot contain `<button>`)
