## Technical Notes

### Séquence d'animation Pick

```
1. PICK (clic sur tâche sidebar)
   │
   ├─► Immédiat:
   │   - TaskTile passe en état "picked" (style planifié + pulse)
   │   - Curseur body → grabbing
   │
   ├─► Animation 300ms ease-out:
   │   - Scroll horizontal vers colonne cible
   │   - Colonnes non-cibles: opacity 0 → pointer-events: none
   │
   └─► État final:
       - Seule colonne cible visible avec ring vert
       - Ghost tile WYSIWYG + lignes précédence
```

### Séquence Place/Cancel

```
2. PLACE (clic sur colonne) ou CANCEL (Escape/clic hors grille)
   │
   ├─► Immédiat:
   │   - Curseur body → default
   │   - TaskTile: état normal (place) ou reset (cancel)
   │
   ├─► Animation 200ms:
   │   - Colonnes non-cibles: fade in (opacity 1)
   │
   └─► Position scroll:
       - PLACE: reste sur colonne actuelle
       - CANCEL: retour position scroll d'origine
```

### États TaskTile Sidebar

| État | Fond | Border-left | Heure | Curseur | Animation |
|------|------|-------------|-------|---------|-----------|
| Non planifiée | Transparent | Non | - | `grab` | - |
| Non planifiée (hover) | Transparent | Non | - | `grab` | shadow + translateY(-2px) |
| **Picked** | Couleur job | Oui (4px) | `...` | - | Pulse opacity 0.7↔1.0 |
| Planifiée | Couleur job | Oui (4px) | `HH:MM` | `default` | - |

### Padding dynamique pour scroll

```css
.grid-container::after {
  content: '';
  min-width: calc(100vw - 240px);
  flex-shrink: 0;
}
```

Permet de scroller n'importe quelle colonne vers une position fixe à gauche du viewport.
