# v0.6.5 - Collapsible Tool Results in AI Chat

**Status:** In Progress
**Branch:** `feature/mcp-llm-scheduler`
**Date:** 2026-03-28

## Summary

Tool execution results (verbose JSON from API responses) are now collapsed by default behind clickable badges. The chat panel shows a short summary ("3 actions executed.") with individual tool badges that expand to show raw details on click.

## Problem

When the AI executed actions (e.g., `extend_task_and_replan`), each result generated a message like `"Executed extend_task_and_replan: {120 lines of JSON...}"`. This flooded the chat panel with unreadable noise.

## Solution

### Backend: Clean summaries instead of JSON dumps
**File:** `services/mcp-server/src/routes/chat.ts`

The confirm endpoint now produces a single summary message (e.g., "3 actions executed. 1 failed.") with all tool calls attached as structured data. Previously it generated one message per action with raw JSON in the content.

### Frontend: Collapsible tool badges
**File:** `apps/web/src/components/Chat/ChatToolCallBadge.tsx`

Tool call badges are now interactive:
- **Collapsed (default)**: Shows status icon + tool label + chevron indicator
- **Expanded (on click)**: Reveals a scrollable, monospace pre-formatted view of the raw result JSON
- Max height: 192px with overflow scroll
- Styled with `bg-zinc-950` dark background, 11px font

The result is a clean, compact chat where operators see what happened at a glance and can drill into details when needed.
