## Technical Notes

### REQ-01: Date Tooltip Implementation

Add a `title` attribute to each DateCell showing the full date in French:

```tsx
// DateCell.tsx
const fullDateString = date.toLocaleDateString('fr-FR', {
  weekday: 'long',
  day: 'numeric',
  month: 'long',
  year: 'numeric',
});
// Result: "vendredi 9 janvier 2026"

// Capitalize first letter
const formattedDate = fullDateString.charAt(0).toUpperCase() + fullDateString.slice(1);

return (
  <button
    type="button"
    title={formattedDate}  // Tooltip on hover
    className="..."
    ...
  >
    ...
  </button>
);
```

This is the simplest approach - no layout changes, uses native browser tooltip.

### REQ-02: Clickable Date Implementation

Modify JobInfo to accept an onClick handler and make the date clickable:

```tsx
// JobInfo.tsx
export interface JobInfoProps {
  job: Job;
  onDateClick?: (date: Date) => void;  // NEW
}

// In render:
<InfoField
  label="Départ"
  value={formatDate(job.workshopExitDate)}
  onClick={() => onDateClick?.(new Date(job.workshopExitDate))}
  isClickable={!!onDateClick}
/>
```

The `onDateClick` will be wired in App.tsx to call `setFocusedDate()`.

### REQ-06: Improved Tooltip

Update StationHeader tooltip to French:

```tsx
title={isCapacityExceeded
  ? `${groupCapacity.groupName} : capacité dépassée (${groupCapacity.currentUsage}/${groupCapacity.maxConcurrent})`
  : `${groupCapacity.groupName} : ${groupCapacity.currentUsage} actifs / max ${groupCapacity.maxConcurrent} en parallèle`
}
```
