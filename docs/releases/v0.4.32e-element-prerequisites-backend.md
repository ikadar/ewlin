## Overview

Backend API support for element-level prerequisites. This release adds PHP API changes to support the frontend work done in v0.4.32a–v0.4.32c.

---

## Prerequisites

- [x] `v0.4.32a` released (Frontend: types, mock data, UI)
- [x] `v0.4.32b` released (Scheduler tile blocking visual)
- [x] `v0.4.32c` released (Forme status & date tracking)
- [ ] `v0.4.32d` released (Documentation)

---

## Scope

### In Scope

**Backend (services/php-api):**
- Add prerequisite fields to Element entity (paperStatus, batStatus, plateStatus, formeStatus)
- Add date tracking fields to Element entity
- Remove prerequisite fields from Job entity
- Database migration for schema changes
- Update API endpoints (GET/POST/PUT for jobs and elements)
- Update OpenAPI documentation

### Out of Scope

- Frontend changes (completed in v0.4.32a)
- Scheduler tile blocking visual (completed in v0.4.32b)
- Forme status for die-cutting (completed in v0.4.32c)
- Documentation (completed in v0.4.32d)
