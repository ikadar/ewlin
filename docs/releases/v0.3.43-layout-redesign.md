## Technical Notes

### REQ-07: Layout Restructure

**Current structure (App.tsx):**
```tsx
<div className="h-screen flex flex-col">
  <TopNavBar />  {/* Full width */}
  <div className="flex-1 flex">
    <Sidebar />  {/* Starts below nav bar */}
    <JobsList />
    ...
  </div>
</div>
```

**New structure:**
```tsx
<div className="h-screen flex">
  <Sidebar />  {/* Full height, with icons at bottom */}
  <div className="flex-1 flex flex-col">
    <TopNavBar />  {/* Width: 100% - sidebar */}
    <div className="flex-1 flex">
      <JobsList />
      ...
    </div>
  </div>
</div>
```

### REQ-07.3: Sidebar Icon Section

Add a bottom section to Sidebar for User/Settings icons:
```tsx
<nav className="w-14 h-full flex flex-col">
  {/* Top navigation */}
  <div className="flex-1 flex flex-col items-center py-3 gap-2">
    <SidebarButton icon={LayoutGrid} ... />
    <SidebarButton icon={Calendar} ... />
  </div>

  {/* Bottom section - settings/user (REQ-07.3) */}
  <div className="flex flex-col items-center py-3 gap-2 border-t border-white/5">
    <SidebarButton icon={Settings} ... />
    <SidebarButton icon={User} ... />
  </div>
</nav>
```

### REQ-08: Add 25% Zoom Level

In `constants.ts`:
```typescript
export const ZOOM_LEVELS = [
  { label: '25%', pixelsPerHour: 20 },  // NEW
  { label: '50%', pixelsPerHour: 40 },
  ...
];
```
