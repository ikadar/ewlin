# v0.6.0 - AI Scheduler Assistant (MCP Server)

**Status:** In Progress
**Branch:** `feature/mcp-llm-scheduler` (all repos)
**Date:** 2026-03-28

## Summary

Adds a natural language interface to the scheduling system. Operators can describe scheduling changes in plain language (e.g., "job 4428 on G37 will be 2h late"), and an LLM (Claude) orchestrates the corresponding API calls.

## Architecture

```
React Chat UI  →  MCP Server (Express/TS, port 3002)  →  PHP API (Symfony)
                       ↕
                  Claude API (tool_use)
```

- **MCP Server** (`services/mcp-server/`): Express + Anthropic SDK service
- **Chat UI**: Sliding right panel in React app, toggled via sidebar or Alt+I
- **Confirmation flow**: Read tools execute immediately; write tools require user confirmation
